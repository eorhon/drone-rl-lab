# drone-rl-lab

Independent research on reinforcement learning for robotic control and sim-to-real autonomy, using PX4 SITL as the primary simulation environment.

## Scope (Phase 1)
**Goal:** Build a reproducible experimental pipeline to study *adaptation under dynamics mismatch* in aerial control.

Initial focus:
- PX4 SITL + Gazebo
- Clean experiment isolation & reproducibility
- Baselines first (classical + RL)
- Publishable results later

## Research Protocol (Before Any Experiment)
Every change/experiment must answer:
1) What is the hypothesis?  
2) What is the measurable metric?  
3) What is the baseline?  
4) What variable am I isolating?  
5) What does failure mean?
